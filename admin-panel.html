          Cybersecurity Lesson

          What Just Happened?

          This demo collected: Personal identification, contact information, birth details, location data, professional information, behavioral patterns, and device fingerprints.

          Real-World Risks

          • Identity Theft: Name + DOB + location = identity theft risk
          • Phishing: Email + name = targeted phishing campaigns
          • Financial Fraud: Profession + concerns = financial profiling
          • Emotional Manipulation: Personal concerns = targeted manipulation
          • Data Resale: All data = valuable commodity on dark web

          Protective Measures

          1. Use unique passwords for every website
          2. Enable two-factor authentication (2FA)
          3. Check privacy policies before submitting data
          4. Use a VPN for sensitive browsing
          5. Regularly monitor your digital footprint
          6. Be skeptical of "free" offers requiring personal data

          Educational Project: This demonstration shows how easily personal data can be collected. Always be cautious online.
