# Bhavishyani - Cybersecurity Awareness Project

A responsive educational website demonstrating how personal data can be collected through astrology and palm reading services, with a focus on cybersecurity awareness.

## Features

1. **Responsive Design**: Works on all device sizes
2. **Interactive Form**: Collects birth details and palm photos with validation
3. **Personalized Reading**: Generates Vedic astrology insights based on input
4. **Photo Upload**: Camera access for palm reading analysis
5. **Educational Reveal**: Shows exactly what data was collected
6. **Admin Panel**: View all collected submissions
7. **Modern UI**: Attractive design with animations and gradients

## How to Use

1. Open `index.html` in a web browser
2. Fill in the birth details form
3. Upload a palm photo (optional)
4. Click "Reveal My Free Destiny Report"
5. View the personalized astrology reading
6. Click "See What Data Was Collected" for educational insights
7. Open `admin-panel.html` to view all submissions

## Files

- `index.html` - Main deceptive astrology website
- `style.css` - Styling and responsive design
- `script.js` - Form handling and data collection
- `admin-panel.html` - Admin dashboard to view collected data
- `admin-style.css` - Admin panel styling
- `admin-script.js` - Admin panel functionality
- `README.md` - This documentation file

## Educational Purpose

This website is created for educational cybersecurity demonstrations to show how personal data can be collected through seemingly harmless services. All data is stored locally in your browser and no actual personal information is transmitted to external servers.

## Key Demonstrations

- How websites use emotional appeals to collect data
- What types of personal information can be gathered
- How biometric data (photos) can be collected
- Fake trust indicators and testimonials
- Data breach simulation and risks

## Technologies Used

- HTML5
- CSS3 (with Flexbox and CSS Grid)
- JavaScript
- Local Storage API
- Font Awesome icons
- Google Fonts

## Disclaimer

This is an educational demonstration only. The astrology readings are generated algorithmically and are not based on actual Vedic astrology calculations. No real personal data is transmitted or stored externally. The website is designed to raise awareness about data collection practices and cybersecurity risks.
